## ✨ Temel Özellikler

* **🏋️ Strength Tracker:** Kişiselleştirilmiş antrenman programları oluşturun ve ilerlemenizi
  kaydedin.
* **🍽️ Meal Planner:** Günlük makro ve kalori hedeflerinize uygun öğün planlaması yapın.
* **🔥 Real-time Sync:** Firebase entegrasyonu sayesinde verileriniz tüm cihazlarınızda anlık olarak
  senkronize edilir.
* **🎨 Modern UI:** Jetpack Compose ile oluşturulmuş, akıcı ve tamamen özelleştirilebilir kullanıcı
  arayüzü.
* **📊 İlerleme Analizi:** Gelişiminizi grafiklerle takip edin (Yakında!).

## 🛠️ Teknoloji Yığını (Tech Stack)

Uygulama, en güncel Android geliştirme standartları kullanılarak inşa edilmiştir:

* **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose) - Modern
  deklaratif arayüz.
* **Language:** Kotlin (Coroutines & Flow)
* **Backend:** * **Firebase Authentication:** Güvenli kullanıcı girişi.
    * **Cloud Firestore:** Ölçeklenebilir NoSQL veritabanı.
* **Architecture:** MVVM (Model-View-ViewModel) & Clean Architecture.
* **Dependency Injection:** Hilt / Dagger (Opsiyonel: Eğer kullanıyorsan burayı tut).
